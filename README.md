# University of Colorado Boulder Drupal 11 Installation Profile

## Usage

```
composer require cu-boulder/boulder_profile
```

### Notes

This profile will not work without the corresponding CU Boulder modules that this profile expects to install.  

It's easier to start with the Drupal 11 project template, which can be found here : https://github.com/CuBoulder/sandpoint-d11-project-template
